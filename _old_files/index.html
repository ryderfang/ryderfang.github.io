我是一名 90 后程序猿，也是本家族祖上三百代第一个码农。2014 年毕业于某二流 985 大学计算机科学与技术专业，随大流读完硕士。开始了慢慢变秃却没有变强的程序猿生涯。

第一份工作是 Windows 桌面软件开发，两年半的时间参于了两款浏览器的开发工作，时代久远，印象里只留下了 Trident/Blink 这两个内核代号。曾经对 Windows 逆向浅尝辄止，这种调试方法需要长时间沉浸在汇编代码里，让我对头发很担忧。第一次接触成百上千万行 C++ 代码的工程，在高配固态硬盘的机器上编译仍需要花上半个小时以上，这样的体验以后也应该不会再有了。

随着 PC 时代的落寞，在第二家公司由 QT 开发转型到 iOS 开发，花了大半个月工资买了台 Macbook Pro 和 一本砖头一样的 《iOS 开发指南》就这么开始了。感谢老东家给的这次转型的机会，让我不至于被拍死在沙滩上。

后来，机缘巧合来到口碑，算了进入了『阿里大家庭』，在这里呆的越久，越觉得自己的弱鸡。这个 BU 独特的优势，能接触到蚂蚁和集团的大部分技术，技术文章多到让人无所适从。对知识过度输入而又吸收无力的焦虑感，越来越明显。

这个博客不会有内部的技术分享，只是我个人技术体系的构建和薄弱基础的补充。
